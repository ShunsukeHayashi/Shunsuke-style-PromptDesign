# Shunsuke-shiki-PromptDesign
シュンスケ式「プロンプトデザイン」

このリポジトリは、シュンスケ式のプロンプトデザインを共有し、異なるタスクやコンセプトに対するChatGPTのプロンプトを探求する場所です。

## シュンスケ式プロンプトデザインについて

シュンスケ式プロンプトデザインは、シュンスケさん独自のアプローチでChatGPTを活用する方法です。この手法は、タスク指向のプロンプトよりも対話やクリエイティブな用途に焦点を当て、独創的な応用方法を追求します。

## 使い方

1. **プロンプトの共有**: シュンスケ式のプロンプトデザイン例を `prompts/` ディレクトリ内にMarkdownファイルとして共有しましょう。

2. **プロンプトの探求**: 他のデザイン例を見て、独自のプロンプトデザインを試してみましょう。シュンスケ式のアイデアを使って、新たなアプローチを追求してみるのも良いですね。

## プロンプトデザインのフォーマット

プロンプトデザインの例を共有する際には、以下のフォーマットに従って記述してください。

```markdown
## テーマ: [プロンプトデザインのテーマ]

**アプローチ**: プロンプトデザインのアイデアやコンセプトの概要
**例**:
- プロンプト: 具体的なプロンプトの例
- 応答: ChatGPTの想像される応答例
